# Typing Foundations (with-typing)

Subapp jugable de `Typing Foundations` (B1) para la parte trasera de la libreta.
Incluye progresión por familias con subniveles (F1–F6) y desbloqueo incremental.

## Run Locally

**Prerequisites:** Node.js 20+


1. Install dependencies:
   `npm install`
2. Run local dev:
   `npm run dev`
   (abre en `http://localhost:3000`)
3. (Opcional) Exponer en LAN:
   `npm run dev:lan`

No requiere `GEMINI_API_KEY` para jugar localmente.
